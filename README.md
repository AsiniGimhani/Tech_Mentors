
# Tech Mentor Website

This is a responsive website for **Tech Mentor Institute**, a higher education platform.

## Pages Included
- **Home** – Welcome page with gallery and intro text  
- **About** – Information about the institute  
- **Courses** – List of courses offered  
- **Faculties** – Profiles of faculty members  
- **Contact** – Contact form to send messages  

## Features
- Responsive design for desktop and mobile  
- Simple navigation bar  
- Contact form ready to connect to a database  
- Login system (optional, PHP & MySQL)  

## Technologies Used
- HTML  
- CSS  
- PHP  
- MySQL
